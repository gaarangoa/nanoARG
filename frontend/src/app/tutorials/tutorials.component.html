Resources used by NanoARG

Computational Tools

Here is a list of the different programs used by NanoARG for doing the computations:
  • Diamond: Accelerated BLAST compatible local sequence aligner.
  • DeepARG: A deep learning based approach for predicting antibiotic resistance genes from metagenomics data.
  • Centrifuge: Is a novel microbial classification engine that enables rapid, accurate, and sensitive labeling of reads and quantification of species on desktop computers.
  • ProSite: Online tool for drawing domain arrangements
  • Cytoscape.js: A javascript library for network visualization.

Databases

  • ARGminer/DeepARG: Antibiotic Resistance Database.
  • NanoARG-MGE database: Mobile Genetic Elements database developed in NanoARG.
  • BacMet: Metal Resistance Database.
  • Functional Genes: UniRef curated protein database for funcional annotation.
  • Pathogens: List of pathogens that are critical to human health from the World Health Organization and the ESKAPE database.
