Relative abundance visualization in R¶

This tutorial will show you how to process output data from the multiple sample comparison table from nanoARG

The goal of this tutorial is to show you how to:

  • Load relative abundances table into R
  • Filter samples of interest
  • Filter categories of interest (MGEs, MRGs, ARGs, etc)
  • Make barplots using relative abundance (counts/Gbp)

Required libraries in R¶

Please make sure to have installed these libraries in your machine

Here we are loading the relative_abundances.tsv file downloaded from nanoARG into our environment.

x = data.table(read.table('relative_abundances.tsv', sep='\t', header=TRUE, quote=NULL))

  • 9: Critial important pathogens
  • 10: Taxonomy

Visualize ARGs relative abundances¶

In this section well show you how to make the relative abundance barchart for MGEs. Note that we will use the index from the Category.

Get only ARGs¶

By setting up the Section to 2 we are able to get only the data related to ARGs.

In [4]:
args = x[x$Section==1,]

In this example we are only processing 4 samples (LSS, HIU, GEM and IND_AS).

In [5]:
args = args[args$Sample=="LSS" | args$Sample=="HIU" | args$Sample=='GEM' | args$Sample=='IND_AS' ]
data = args[, list(sum=sum(counts.Gbp)), by=list(Sample, Category)]

Define the order you want the samples in the x axis

In [6]:
clev = c("IND_AS", "GEM", "HIU", "LSS")

Get data ready for visualization

In [7]:
data$Sample = factor(data$Sample, levels=clev)

# order by sum of elements
datap = args[, list(sum=sum(counts.Gbp)), by=list(Sample, Category)]
dtr = xtabs(sum~Sample+Category, data=datap)
dtr = colSums(dtr)
lev = names(sort(dtr, decreasing=TRUE))

data$Category = factor(data$Category, levels=lev)

Make the stacked barchart¶

In [8]:
ggplot(data)+geom_bar(aes(Sample, y=sum, fill=Category ), colour="black", stat='identity', size=0.25)+theme_bw() +
scale_fill_manual(
    values = c(
        diverge_hcl(15, h=c(260,0)),
        terrain_hcl(15, c=c(40, 80)),
        heat_hcl(7)
    )
) +
theme(axis.text.x = element_text(angle=90, hjust=1, vjust=0.5))

Remarks¶

To process any other category such as MGEs, MRGs, etc. modify this line accordingly to the Section you want to analyze:

args = x[x$Section==1,]
